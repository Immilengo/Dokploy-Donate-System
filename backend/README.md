# Backend

API do sistema de doações.

## Tecnologias

- Node.js
- Express
- Prisma
- PostgreSQL
- TypeScript

## Scripts

```bash
npm run dev
npm run build
npm run start
npm run test
npm run prisma:generate
npm run prisma:migrate
npm run prisma:deploy
npm run seed
```

## Estrutura

- `src/`: código-fonte da API
- `prisma/`: schema, migrations e seed
- `tests/`: testes automatizados
