# Frontend

Interface web do sistema de doações.

## Tecnologias

- Next.js 14
- React 18
- TypeScript
- Tailwind CSS
- shadcn/ui

## Scripts

```bash
npm run dev
npm run build
npm run start
npm run lint
```

## Estrutura

- `src/app/`: rotas e páginas
- `src/components/`: componentes de UI e layout
- `src/services/`: integração com a API
- `src/store/`: estado global
